Descripción del Proyecto

Este proyecto corresponde al desarrollo y simulación de un Autómata con Pila (PDA) en entorno web, como parte del curso de Autómatas y Lenguajes Formales. El objetivo es diseñar un autómata capaz de reconocer un lenguaje no regular, implementando su funcionamiento mediante una página web interactiva.

Lenguaje: L = { aⁿ bᵐ x y* cᵐ aⁿ | n ≥ 2, m ≥ 1 }

Objetivo General

Desarrollar una simulación web funcional que represente el comportamiento de un Autómata con Pila (PDA) diseñado para reconocer el lenguaje L, permitiendo el ingreso de cadenas, la visualización de la pila y la verificación de aceptación o rechazo de la cadena ingresada.

Descripción Teórica del PDA

    

      
ElementoDescripción

      
Σ (Alfabeto){ a, b, c, x, y }

      
Γ (Símbolos de pila){ A, B, Z }

      
Q (Estados){ q₀, q₁, q₂, q₄, q₅ }

      
q₀Estado inicial

      
Z₀Símbolo inicial de pila

      
F{ q₅ } (estado de aceptación)

    


    

      El autómata empuja “A” por cada ‘a’, luego “B” por cada ‘b’. Al llegar a
      “x” entra en el estado de transición y lee los posibles “y”. Posteriormente,
      por cada “c” saca un símbolo “B” de la pila y finalmente elimina las “A”
      cuando aparecen las últimas “a”. Si la pila queda con solo Z, la cadena
      es aceptada.
